## Featured Project

### codex-workflow-kit

An open-source workflow template for using Codex and Claude Code in real projects.

It is built around a simple idea:

AI coding becomes much more useful in production work when the workflow is stable, controllable,
and reusable.

What’s inside:

- `AGENTS.md` / `CLAUDE.md` entry setup
- memory templates
- default skills for task capsule, high-risk preflight, and handoff
- specialized skills for repeatable workflows
- validation and maintenance rules
- example project shells
